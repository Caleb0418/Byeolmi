# 사용자 여정 지도 (User Journey Map) & 핵심 유저 여정 (CUJ)

봉봉 마켓(BongBong Market)의 성공을 결정짓는 핵심 사용자 경험 단계들을 진단하고, 솔루션 검증 단계에서 **반드시 성공(Wow Experience)해야 하는 핵심 유저 여정(Critical User Journey, CUJ)**을 규명합니다.

---

## 1. 사용자 여정 지도 (User Journey Map)

사용자가 인지 단계부터 최종 정산 및 재주문에 이르기까지 느끼는 감정 곡선과 터치포인트를 정리했습니다.

| 여정 단계 | 1. 온보딩 및 인지 | 2. 주문하기 (Order) | 3. 달성도 모니터링 | 4. 마감 및 정산 (Settlement) |
| :--- | :--- | :--- | :--- | :--- |
| **고객 여정** | 카카오톡 채널 추가 유도 메시지를 받음. | 카톡 채팅창 내에서 주문서를 띄워 수량을 입력함. | 실시간 단가 구간과 남은 박스 수를 조회함. | 최종 단가 알림톡을 받고 정해진 계좌로 송금함. |
| **터치포인트** | 카카오 알림톡 / 문자 메시지 | 카톡 채널 챗봇 / 간편 웹뷰 | 모바일 웹뷰 그래프 | 카카오 정산 알림톡 / 송금 화면 |
| **사용자 감정** | "새로운 방식이네? 귀찮진 않을까?" (주의) | "어, 복잡한 가입 없이 그냥 카톡에서 열리네?" (안도) | "와, 벌써 50박스 모였네? 조금 더 할인되겠다!" (흥미) | "내가 입금할 금액이 딱 정리돼서 오니 편하네." (만족) |
| **감정 지수** | 😐 (보통) | 🙂 (긍정) | 😄 (매우 긍정) | 🥰 (최고) |
| **해결 기회** | 한눈에 장점을 알 수 있는 간결한 카피 제공 | 입력 및 로딩 최소화, 주소 자동 저장 기능 | 직관적인 게이지 그래프 디자인, 지연 없는 반영 | 원클릭 계좌 복사 및 이체 링크 연동 |

---

## 2. 핵심 유저 여정 (CUJ: Critical User Journey)

서비스 존속과 고객 유지(Retention)를 위해 **시스템상 100% 깔끔하게 작동하고 사용자에게 감동을 주어야 하는 3대 핵심 경험**입니다.

### 🎯 CUJ 1: 앱 설치 없는 모바일 주문 완결 (Frictionless Order)
* **상황**: 차량 이동이 잦은 구매자(업자)가 도매점 주문 마감 30분 전에 급하게 주문을 넣어야 하는 상황.
* **성공 기준**: 
  1. 카카오톡 채팅방 하단 메뉴의 **[간편 주문하기]**를 터치한 지 **2초 이내**에 주문 화면이 뜹니다.
  2. 아이디/패스워드 입력 없이 기존 정보를 바탕으로 바로 로그인 상태가 유지됩니다.
  3. 품목 선택과 수량 입력 후 **[주문 완료]** 버튼을 누르면, **5초 이내**에 카카오톡 대화방으로 주문서 요약 알림 메시지가 도착해 정상 접수되었음을 신뢰하게 만듭니다.

### 🎯 CUJ 2: 누적 수량에 따른 실시간 단가 하락 정보의 투명한 인지 (Interactive Sliding Pricing)
* **상황**: 업자가 오늘 공구에 참여하면서 "조금만 더 많이 구매되면 한 단계 아래 할인 가격으로 납품받을 수 있을 텐데..."라고 생각하며 주문 조율을 고민하는 상황.
* **성공 기준**:
  1. **[공구 현황판]** 클릭 시 현재까지 누적된 주문 총량이 눈에 띄는 **진척도 게이지(Progress Bar)**로 노출됩니다.
  2. 다음 할인 가격까지 남은 목표 박스 수가 시각적으로 뚜렷하게 보입니다 (예: *"20박스 추가 달성 시, 전원 2,000원 추가 할인!"*).
  3. 이 시각적 요소를 통해 구매자 스스로 구매량을 조정하거나 다른 파트너 업자에게 공구 참여를 독려하는 자연스러운 바이럴이 일어나야 합니다.

### 🎯 CUJ 3: 마감 후 1분 이내 정산 안내 및 스마트 송금 (Automated Closed Loop Settlement)
* **상황**: 오후 5시 마감 시간, 사장님은 정산 계산기를 두드려 수작업으로 청구서를 돌려야 하고, 업자는 정확히 얼마를 어디로 보내야 하는지 기다리는 상황.
* **성공 기준**:
  1. 사장님이 관리자 대시보드에서 **[일괄 마감 및 알림 발송]** 버튼을 클릭하는 순간, **1분 이내**에 모든 구매자에게 최종 단가로 계산된 정산 금액이 개인 카톡으로 전송됩니다.
  2. 구매자는 정산 메시지에 포함된 **[복사하기]**나 **[송금하기]** 링크를 터치해 은행 앱에서 계좌번호나 금액을 수기 타이핑하지 않고 **터치 한 번으로 정확한 송금**을 마칠 수 있어야 합니다.
