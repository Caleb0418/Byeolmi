# 품목 이미지 동적 등록 및 실시간 연동 계획서

사장님이 품목 등록 시 원하는 이미지의 웹 URL 또는 로컬 경로를 직접 입력하고, 이를 구매자 주문서 화면에 실시간으로 반영하기 위한 프론트엔드 연동 계획입니다.

---

## 1. 개요
* **목적**: 상품 사진을 사장님이 수시로 업데이트하여 구매자(업자)가 신선도나 상품 형태를 직관적으로 파악할 수 있도록 신뢰성을 강화합니다.
* **동작 원리**: 
  1. 사장님이 품목 관리 모달에서 이미지의 경로(예: 로컬 내 `fresh_vegetables.png` 또는 외부 웹 주소 `https://...`)를 입력하고 저장합니다.
  2. `app.js` 내에 데이터가 갱신되어 브라우저 `localStorage`에 저장됩니다.
  3. 구매자 화면(`client.html`)의 렌더링 카드 내 이미지 `src`가 사장님이 등록한 이미지 경로로 바인딩되어 실시간 출력됩니다.

## 3. 검증 시나리오
* **이미지 연동 테스트**:
  1. 대시보드 모달에서 신규 품목 추가 시 이미지 주소에 웹상의 임의의 사진 주소(예: `https://images.unsplash.com/photo-1518977676601-b53f82aba655?w=200` 등 감자 사진 주소)를 입력하고 저장합니다.
  2. 왼쪽 주문자 웹뷰에 해당 웹 이미지 사진이 깨짐 없이 깔끔하게 실시간 렌더링되는지 확인합니다.
