## 2. 권한 위임 (멤버 초대) — 비밀번호 대신 이렇게

사장님은 각 사이트에서 **초대 버튼만** 눌러주시면 됩니다. (초대받을 계정은 저희가 알려드립니다.)

| Supabase / Vercel | 서버·배포 | (저희가 관리 중) 필요 시 멤버 초대 |

## 4. 비용 안내 (카카오 알림톡 / Solapi)

선불 충전식 + **발송 건당 과금**이며, 월 기본료는 없습니다. 월 발송량 **1만 건 미만**
구간은 건당 **13원(VAT 별도)** 입니다.

| 월 발송량 | 건당 단가 | 월 예상 (VAT 별도) | 월 예상 (VAT 포함) |
|---|---|---|---|
| 50건 | 13원 | 650원 | **715원** |
| 100건 | 13원 | 1,300원 | **1,430원** |
| 200건 | 13원 | 2,600원 | **2,860원** |

- 위 단가는 **월 1만 건 미만 기준**입니다. 물량이 늘면 단가가 내려갑니다
  (1만 건↑ 10원 / 10만 건↑ 8원).
- 별미집은 주문 1건당 알림톡이 최대 2종(**발주 확인 + 정산 명세서**) 발송될 수 있어,
  위 "건수"는 **주문 수가 아니라 메시지 수** 기준으로 보시면 됩니다.
  (예: 하루 평균 3~7건 주문 시 월 100~200건 수준)
- 알림톡 발송 실패 시 문자(SMS/LMS)로 대체발송하도록 설정하면 해당 문자 단가는 별도입니다.
- 충전·결제는 **사장님 Solapi 계정**에 묶입니다(저희 계정으로 대신 결제 ❌).
- 단가는 변동될 수 있어, 가입 시 [Solapi 가격 정책](https://solapi.com/pricing)에서
  최종 확인을 권장합니다.

## [별첨 · 선택] 세금계산서 자동 발행 (볼타) — 적용을 원하실 경우에만

> 💡 명세서(거래·입금 안내)와 **전자세금계산서(국세청 발행)는 별개**입니다.
> 세금계산서 자동화는 **적용 여부를 결정하신 뒤 별도 트랙**으로 진행하며, 코드도 추후
> 추가됩니다. 적용을 원하실 경우 아래 정보가 추가로 필요합니다.

| 항목 | 비고 |
|---|---|
| 볼타 계정 | 멤버 초대 방식 + 발급된 API 키 |
| 과세사업자 여부 / 업종·업태 | 발행 주체 확인 |
| 공급자 정보(상호·사업자번호·대표자·주소) | 세금계산서 기재용 |
| 볼타 요금제 | 건당/월정액 등 발생 비용 확인 |

### 처리 순서 요약
1. **위 1번 정보**를 정리해 전달
2. **위 2번 사이트**에서 저희 계정을 멤버로 초대
3. **위 3번 인증**(사업자·발신번호)을 직접 처리
4. (선택) 세금계산서를 원하시면 별첨 정보 전달 → 별도 일정으로 진행
