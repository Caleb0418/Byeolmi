# 별미집 (BongBong) - 도매 발주 및 정산 시스템

신선식품 도매 공급자(사장님)가 거래처의 발주를 취합하고, 수량별 차등 단가를 적용해 정산한 뒤, 카카오 알림톡으로 명세서를 발송할 수 있도록 돕는 B2B 정산 관리 도구입니다.

## 1. 실행 방법

본 프로젝트는 순수 HTML, CSS, JavaScript로 구성되어 있으며 별도의 빌드 과정 없이 웹 브라우저에서 바로 실행할 수 있습니다.

- **사장님용 대시보드**: [index.html](file:///c:/Users/seung/Desktop/유승종/11_BongBong/index.html)을 웹 브라우저로 엽니다.
- **구매자용 발주서**: [client.html](file:///c:/Users/seung/Desktop/유승종/11_BongBong/client.html)을 웹 브라우저로 엽니다.
- **정산 명세서 (영수증)**: [invoice.html](file:///c:/Users/seung/Desktop/유승종/11_BongBong/invoice.html)을 웹 브라우저로 엽니다.

## 2. 파일 구성 및 역할

- **`index.html`**: 사장님이 실시간 발주 현황을 파악하고, 단가를 조절하며, 정산 알림톡을 발송할 수 있는 관리자용 대시보드 화면입니다.
- **`client.html`**: 구매자(도매상, 마트 등)가 모바일 또는 데스크톱에서 당일 공급 품목 리스트를 확인하고 발주를 제출하는 주문용 화면입니다.
- **`invoice.html`**: 사장님이 발주를 마감하고 단가를 확정하면, 구매자가 카카오 알림톡 링크를 통해 접근하여 최종 정산 내역 및 송금 계좌를 확인하는 모바일 명세서 페이지입니다.
- **`app.js`**: 전체 애플리케이션의 상태 관리, 비즈니스 로직, 데이터 입출력(Supabase 연동)을 처리하는 코어 스크립트입니다.
- **`legacy/`**: 디자인 프로토타입 및 이전 버전의 HTML/에셋 등을 안전하게 보관하는 격리 디렉토리입니다.

## 3. 현재 개발 상태

- 기존 LocalStorage 기반의 1회성 목업 환경에서 **진짜 DB(Supabase) 연동 단계**로 마이그레이션이 진행 중입니다.
- 데이터 공유가 불가능했던 문제를 해결하기 위해 실시간 데이터베이스 동기화 및 보안(RLS) 정책을 반영하고 있습니다.
